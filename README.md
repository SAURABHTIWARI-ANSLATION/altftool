# AltF Tools Standalone

This repository contains the standalone tools engine for AltF. It is designed to be deployed independently (e.g., on GitHub Pages or Vercel) and embedded via iframes in the main application.

## Features
- Over 100+ high-performance tools.
- Minimal, transparent layout optimized for iframe embedding.
- No global headers, footers, or ads (ads should be placed around the iframe in the host app).

## Tech Stack
- Next.js (App Router)
- Tailwind CSS
- Framer Motion & Lucide Icons

## Setup
```bash
npm install
npm run dev
```
